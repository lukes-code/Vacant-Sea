# Vacant Sea

https://vacant-sea.vercel.app/

Requires API key and space to run locally.

## Technologies

- Next (functional React using context)
- TypeScript
- GraphQL (Contentful Headless CMS)
- Tailwind
- Radix-UI
- Framer motion
